# Analytical Data Platform – Hacker News

## Overview

Plataforma analítica de datos construida sobre información pública de **Hacker News** con arquitectura tipo *medallion*. El sistema ingiere datos semi-estructurados desde la API oficial, los organiza en capas y expone modelos analíticos optimizados para consultas sobre contenido, usuarios y engagement.

**Stack:** Python 3.13+ · Pandas · PyArrow (Parquet) · Airflow
